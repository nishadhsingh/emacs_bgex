# Emacs-BGEX patch とは?

X 用 (X なら Cygwin でも MacOSX でも動きます!) FSF Emacs の背景拡張非公式パッチです。

Emacs-BGEX patch を適用すると背景に画像や色を指定できるようになります。

フレーム単位ではなく、バッファごとに異なる背景を指定することもできます。

かれこれ 10 年以上メンテナンスしています。
